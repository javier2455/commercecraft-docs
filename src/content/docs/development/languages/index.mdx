1. **File: `project.config.ts`**

- **Purpose:** Define the languages available on the site.

- **Task:** Add new languages to the localization mapper.

```typescript
const localizationMapper = {
 en: {
    locale: 'en_US',
    currency: 'USD',
    currencyCode: '$',
    countryCode: 'US',
    countryName: 'United States',
    },
 fr: {
    locale: 'fr_FR',
    currency: 'EUR',
    currencyCode: '€',
    countryCode: 'FR',
    countryName: 'France',
    },
 it: {
    locale: "it_IT",
    currency: "EUR",
    currencyCode: "€",
    countryCode: "IT",
    countryName: "Italy",
  },
} as Record<string, LocalizationMapping>;
```
