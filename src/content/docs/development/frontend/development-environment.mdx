### Errores comunes

Si durante la instalación reciben algunos de los siguientes errores a continuación se explica posibles vías de solución:

#### Error

Couldn't find manifest for 'frontastic-cli'

#### Solución

Verificar a través del comando: scoop bucket list, que se encuentre el registro del archivo:

"FrontasticGmbH_scoop-bucket **https://github.com/FrontasticGmbH/scoop-bucket** `fecha de la instalación` en la lista de instalaciones, en caso de que no se encuentre volver a ejecutar el comando:

<Code code={code6} lang="text" title="powershell" />

hasta que se agregue a la lista. En caso de no agregarse ejecutar el comando "scoop bucket add extras" y luego volver a ejecutar "scoop bucket add FrontasticGmbH_scoop-bucket https://github.com/FrontasticGmbH/scoop-bucket.

#### Error

RPC failed; curl 56 Recv failure: Connection was reset

#### Solución

Este error es asociado a la conexión de tu pc. Revisar la conexión, vpn, firewalls, etc.
